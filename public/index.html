API Endpoints

1. List of All Pokemon

Returns the list of all Pokemons.

  • All Pokemons: https://oodaiden-pokemon-server.onrender.com/pokemon

Add Pagination Query parameters for page number, and results per page as follows.

  • Pagination: https://oodaiden-pokemon-server.onrender.com/pokemon?page=1&limit=20

    • Default: page=1 and limit=20 items per page.
      Add query params to change page number and results per page by appending URL as follows.
      Example: URL + /pokemon?page=26&limit=40 (returns page 26 and 40 items per page)

2. Single Pokemon by id (where id = Pokemon id)

Append the Point 1 endpoint with the Pokemon ID, as follows.

  • Pokemon id: https://oodaiden-pokemon-server.onrender.com/pokemon/62

3. Single Pokemon by id/info (where info = "name", "type", or "base" Pokemon charactaristic)

Append the point 2 endpoint with the Pokemon "info" charactartistic, as follows.

  • Pokemon name: https://oodaiden-pokemon-server.onrender.com/pokemon/62/name
  • Pokemon type: https://oodaiden-pokemon-server.onrender.com/pokemon/62/type
  • Pokemon base: https://oodaiden-pokemon-server.onrender.com/pokemon/62/base

4. Query parameters returning result by id, name, type

Append base URL endpoint in point 1 with the Pokemon "id", "name", "type", as follows.

  • Pokemon id: https://oodaiden-pokemon-server.onrender.com/pokemon?id=262

    Returns the Pokemon with the matching id.

  • Pokemon name: https://oodaiden-pokemon-server.onrender.com/pokemon?name=Mightyena

    Returns the Pokemon with the matching name.

  • Pokemon type: https://oodaiden-pokemon-server.onrender.com/pokemon?type=Dark

    Returns an array of all Pokemon with the matching type.
